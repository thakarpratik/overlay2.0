# OverlayTool — Social UI Build

✅ Template browsing UX (search + category chips + live mini previews)  
✅ Brand Kit (font + colors, apply to selected/all)  
✅ Text upgrades (font picker, weight, line-height, letter spacing, shadow, outline)  
✅ 2:3 canvas only (1000×1500)  
✅ IndexedDB storage (no quota issues)  
✅ Export PNG + ZIP (full-res)

## Run
```bash
npm install
npm run dev
```

## Routes
- `/` homepage
- `/templates` marketing template gallery
- `/tool` dashboard
- `/tool/new` upload + template browser
- `/tool/editor/[projectId]` editor
